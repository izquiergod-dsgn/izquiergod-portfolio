# Imágenes de proyectos

Coloca aquí las imágenes para el selector de proyectos (panel derecho).

| Archivo esperado                | Proyecto              |
|---------------------------------|-----------------------|
| palmas-ilustracion.jpg          | Palmas Ilustración    |
| palmas-web.jpg                  | Palmas Web            |
| canciones.jpg                   | Canciones             |
| exploracion.jpg                 | Exploración           |
| tributo-gl.jpg                  | Tributo GL            |
| powerlifting.jpg                | Powerlifting          |

Tamaño recomendado: 1200×800px o mayor, JPG o WebP.
object-fit: cover se encarga del recorte.

Para activar cada imagen, abre src/data/projects.js
y descomenta la línea `img:` del proyecto correspondiente.
